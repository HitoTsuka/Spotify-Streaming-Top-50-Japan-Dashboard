Project Overview

Welcome to my data visualization portfolio. Here you'll find my latest project on Spotify's Top 50 Streaming Songs in Japan, analyzed and visualized using Tableau.

Spotify Streaming Top 50 Japan Dashboard

This dashboard visualizes the top 50 streaming songs in Japan, providing insights into rankings, popularity, and trends across various genres.

Data Analysis Details

  • Data Collection

    Data was collected using the Spotify API, which provides real-time information on streaming counts and song metadata. Since the API only allows for the retrieval of daily data, a process was implemented to accumulate this information over time. This ensures that the analysis includes comprehensive historical data for accurate trend analysis and insights.

  • Data Processing

    Data was processed and cleaned using Python to ensure accuracy and relevance before being visualized in Tableau.

  • Analysis Techniques

    Various analysis techniques were applied, including trend analysis, popularity tracking, and genre comparison.

What You Can Do

  • Track Song Performance

    Click on any of the top 50 songs to view detailed trends in ranking and popularity.

  • Analyze Custom Periods

    Investigate ranking and popularity trends over specific date ranges.

  • Explore Artist Insights

    Analyze the performance of specific artists, observing trends in their rankings and popularity.

  • Compare Genre Trends

    Review and compare trends across different music genres, gaining insights into genre-specific performance.
